Este proyecto es un reproductor de audio hecho en Flutter usando arquitectura BLoC.
Lo que puede hacer:

Reproducir 8 canciones .

Play, pause, siguiente, anterior y mover la barra de progreso.

Cambiar el volumen y la velocidad.

Agregar canciones desde internet mediante URL.

Cambiar de canción deslizando la imagen.

Menú lateral animado.

Pantalla de carga personalizada.
